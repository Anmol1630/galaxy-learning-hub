Student Exam Performance Predictor

Problem → Predict exam outcomes to identify at‑risk students early.

Solution → Cleaned data, engineered features (attendance, study time), trained multiple models, selected the best via cross‑validation, and deployed behind a Flask API.

  • Pipeline: ingestion → validation → training → evaluation → registry
  • Monitoring: basic drift checks + logs
  • Infra: S3/Blob for artifacts; simple CI for tests
PriceWise: Laptop Price Estimator

Problem → Estimate fair market prices given specs.

Solution → Regression with feature crosses (CPU × RAM × GPU tiers), target transformation for skew; exported a lightweight model for web demo.

  • Stack: scikit‑learn, Pandas
  • Evaluation: MAE/RMSE, residual diagnostics
  • Packaging: pickle + versioned artifacts
